# Iris Flower Prediction 🌸

This is a Machine Learning project that predicts the species of iris flowers using Python and Scikit-learn.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy
- Matplotlib

## Features
- Predicts Iris flower species
- Trained using ML algorithms
- Simple and beginner friendly

## Output
The model predicts:
- Iris Setosa
- Iris Versicolor
- Iris Virginica
